El Impostor

Juego de adivinanza social para amigos

📋 Instrucciones del Juego

  1. 🔑
    Configura tu API key

    Necesitarás una API key de Google Gemini para generar las palabras

  2. 👥
    Registra a los jugadores

    Mínimo 3 participantes, ingresa los nombres en orden horario

  3. 🎯
    Revela tu rol

    Cada jugador toca la pantalla para ver si es el impostor o conoce la palabra

  4. 💬
    Comienza la discusión

    El impostor debe descubrir la palabra, los demás deben identificar al impostor

📖 Reglas del Juego

  • 🏆 Ganan los jugadores normales si identifican al impostor
  • 🕵️ Gana el impostor si adivina la palabra secreta
  • ⏰ Tienen 5-10 minutos para discutir y votar
  • 🗣️ Cada jugador puede hacer preguntas y dar pistas
